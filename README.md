


**Counter Application (React)**

A simple and interactive Counter Application built using React. This project demonstrates how to manage state using the useState hook and update the UI dynamically based on user interactions.

🚀** Features**
Increment, Decrement, and Reset functionality
Prevents the counter value from going below zero
Displays a message when the minimum limit is reached
Instant UI updates using React state
Clean and user-friendly interface
🛠️ Technologies Used
React.js
JavaScript (ES6)
HTML
CSS

**💡 Overview
**
This application uses React functional components and the useState hook to handle state changes. User actions such as button clicks trigger updates, and the UI re-renders automatically to reflect the current counter value.

🎯 Learning Outcome
Understanding React functional components
Managing state with useState
Handling events in React
Implementing conditional rendering
